# Technologie brzegowe (Edge AI)

## Opis kursu

Kurs ma charakter projektowy i koncentruje się na budowie kompletnego systemu Edge AI przeznaczonego do zastosowań w monitoringu i bezpieczeństwie (np. budowy, magazyny, obiekty zamknięte).

Praca odbywa się w zespołach 2-osobowych nad jednym wspólnym typem systemu, rozwijanym etapowo przez cały semestr.

System integruje dwa główne źródła informacji:

* obraz z kamery (Computer Vision),
* dane z czujników (Sense HAT),

oraz wykorzystuje modele językowe (NLP / LLM) do wysokopoziomowej komunikacji z użytkownikiem.

## Efekt kursu

Efektem kursu jest działający prototyp zdolny do:

* wykrywania zdarzeń w czasie rzeczywistym,
* analizy danych sensorycznych,
* fuzji informacji z różnych źródeł,
* interpretacji zdarzeń przez lokalny model językowy,
* generowania raportów i odpowiedzi użytkownikowi.
